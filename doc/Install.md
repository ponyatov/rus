# Установка

Система поставляется в виде исходных кодов, которые нужно скопилировать под ОС
Linux (сборка под Windows не поддерживается).

```
$ cd ~
$ git clone -o gh https://github.com/ponyatov/rus
$ cd ~/rus
~/rus$ make install
```

## Компиляция из исходных кодов

```
~/rus$ make
```

## Запуск в эмуляторе

```
~/rus$ make i386
```
